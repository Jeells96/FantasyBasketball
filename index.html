How BGL Scoring Works
01 — LOCK
Every Sunday at 10 AM, your starting lineup locks. Those starters — and only those starters — count for the week. No swaps, no injury replacements.
02 — BEST GAME
Monday through Sunday, each starter plays multiple games. Only their single highest-scoring game counts toward your weekly total.
03 — MATCHUP
Sum each starter's best game. The team with the higher total wins the week. Standings are tracked here, independent of ESPN.

Firebase Setup Required

This app uses Firebase Firestore for multi-device sync. To enable it:

  1. Go to console.firebase.google.com → Create project
  2. Add a Web app → copy the firebaseConfig object
  3. Enable Firestore Database in test mode
  4. Replace the FB_CONFIG block in this file with your config

Without Firebase, data is stored in localStorage on this device only.
